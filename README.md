# botão de pressão LED


## Sobre o projeto

Este projeto foi desenvolvido na plataforma Wokwi com o objetivo de demonstrar o funcionamento de um botão de pressão controlando um LED em um circuito eletrônico simples. O projeto é muito utilizado para iniciantes em eletrônica e programação com Arduino, pois ajuda a compreender conceitos básicos de entrada e saída digital.

Ao pressionar o botão, o LED acende. Quando o botão é solto, o LED apaga. Esse funcionamento simula sistemas presentes em diversos dispositivos eletrônicos do dia a dia.

## Objetivo do projeto

O principal objetivo é ensinar como utilizar um botão de pressão para enviar sinais ao Arduino e controlar um componente de saída, no caso o LED. Além disso, o projeto ajuda no aprendizado sobre montagem de circuitos e lógica de programação básica.

## Componentes utilizados
1 Arduino Uno
1 LED
1 Botão de pressão (Push Button)
1 Resistor para o LED
1 Resistor para o botão
Protoboard
Fios de conexão
Funcionamento

O botão funciona como um interruptor momentâneo. Quando ele é pressionado, o Arduino identifica o sinal elétrico e envia energia para o LED, fazendo-o acender. Quando o botão deixa de ser pressionado, o sinal é interrompido e o LED apaga.




## Conclusão

O projeto “Botão de Pressão com LED” é uma atividade simples e importante para quem está começando a estudar Arduino e eletrônica. Ele permite entender como componentes físicos interagem com a programação, servindo como base para projetos mais avançados no futuro.
